About Routines & Habits

This website was created as a Web and Multimedia Development assignment for Year 1 Semester 2 at Botswana Accountancy College. The goal was to plan, design, and build a fully functional static website using HTML5, CSS3, and Bootstrap, then publish it live using GitHub Pages.

Why This Topic Was Chosen

The topic was chosen because time management and effective studying are challenges that almost every student faces. As a student who has personally struggled with procrastination, inconsistency, and retaining information, this topic felt both relevant and meaningful. Rather than presenting expert advice, the website shares practical strategies that students can experiment with and adapt to their own situations.

Technologies Used

  • HTML5 — for structuring all content using semantic elements such as header, nav, main, footer, section, and aside
  • CSS3 — external stylesheet controlling all visual styling, layout, hover effects, and responsive design
  • Bootstrap 5.3 — loaded via CDN for responsive utilities and base styles
  • JavaScript — Bootstrap's JS bundle loaded before closing body tag for interactive components
  • VS Code — code editor used throughout development
  • Git and GitHub — version control and hosting via GitHub Pages

Development Approach

The website was built following the web development lifecycle — requirements and specifications first, then information design and site mapping, wireframes, HTML structure, CSS styling, and finally testing and publishing. This approach made it easier to catch problems early and keep the work organised across each stage.

Future Plans

Future versions could include interactive tools such as a habit tracker or study schedule generator, content for different types of learners, and a community feature where students can share strategies with each other. These improvements are discussed in detail in the planning documentation.
